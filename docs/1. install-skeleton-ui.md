# Installing Skeleton UI

## Install Skeleton:

See: [Installing Skeleton](https://www.skeleton.dev/docs/svelte/get-started/installation/sveltekit)

```sh
pnpm i -D @skeletonlabs/skeleton @skeletonlabs/skeleton-svelte
```

## Configuring Tailwind

Add to `/src/routes/layout.css`:
```css
@import '@skeletonlabs/skeleton'; 
@import '@skeletonlabs/skeleton-svelte'; 
@import '@skeletonlabs/skeleton/themes/cerberus'; 
```

## Set Active Theme

In `/src/app.html`:
```html
<html data-theme="cerberus">
    <!-- ... -->
</html>
```

## Install Icons
See: [Lucide Svelte](https://lucide.dev/guide/packages/lucide-svelte)

```sh
pnpm add @lucide/svelte
```
